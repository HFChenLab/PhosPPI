PhosPPI Web Server



PhosPPI is a Sequence-based Machine Learning method for predicting the effect of Phosphorylation on Protein-Protein Interactions (PPI). Compare with other competing methods, PhosPPI does not require PDB complex and interfacial phosphorylation sites as input. In addition, PhosPPI achieved better predictive performances (Accuracy and AUC) in two independent validation sets.
Researchers can input two interacting protein sequences and designate one or multiple phosphorylation sites (Serine, Threonine, Tyrosine), this Web Server will return following predictive results, including:
(1)Site-specific Phosphorylation has no effect on PPI; (2)Site-specific Phosphorylation increases the protein-protein affinity (Enhance); (3)Site-specific Phosphorylation decreases the protein-protein affinity (Inhibit).
